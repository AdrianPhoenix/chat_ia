# Chat IA Avanzado

Chat con inteligencia artificial usando Node.js, Express y Tailwind CSS.

## Instalación

```bash
npm install
```

## Uso

```bash
# Desarrollo
npm run dev

# Producción
npm start
```

## Características

- ✅ Backend con Node.js y Express
- ✅ Frontend con Tailwind CSS
- ✅ API de Cohere integrada
- ✅ Diseño responsive y moderno
- ✅ Indicador de escritura
- ✅ Variables de entorno seguras

## Estructura

```
chat-avanzado/
├── server.js          # Servidor Express
├── package.json       # Dependencias
├── .env              # Variables de entorno
└── public/
    ├── index.html    # Frontend
    └── script.js     # JavaScript del cliente
```
